Connect your listings

Your site fetches live listings from your Google Sheet via Apps Script. Here's how to set it up:

One-time setup (5 minutes):
1. Copy appscript-v1.gs into a new Apps Script project
2. Paste your Google Sheet ID into SHEET_ID
3. Run setupSheet() once to create the Listings tab
4. Click Deploy → New deployment → Web app
5. Set "Who has access" to Anyone
6. Copy the deployment URL and paste it below
No sheet yet? Leave this blank — your template will show 3 sample listings as a fallback until you connect a sheet.
